# gmail-promo-cleaner
A no-code Gmail automation workflow built with Zapier to automatically filter and remove promotional emails, helping keep inboxes clean and clutter-free

🚀 Project Overview

I spent hours manually deleting promotional emails every day.
To solve this, I built a 3-step Zapier workflow that automatically filters and removes unwanted emails in the background.
The automation works 24/7 and helps keep the inbox clean without manual effort.

⚙️ Workflow Steps

📩 Step 1 — Gmail Trigger
Detects every new email entering the inbox
Checks inbox every few minutes automatically

🔍 Step 2 — Filter by Zapier
The workflow checks if the sender or subject contains keywords like:
promo
marketing
newsletter
noreply

✅ If matched → Continue workflow
❌ If not matched → Ignore email
🗑️ Step 3 — Gmail Action
Automatically deletes matched promotional emails
No manual clicks required
